# TimeTravel Agency

Web app immersive Next.js pour une agence fictive de voyage temporel de luxe.

## Fonctionnalités

- Landing page premium avec hero animé, ambiance sombre et accents dorés
- Présentation de l'agence avec reveal au scroll
- Galerie interactive de 3 destinations (Paris 1889, Florence 1504, Crétacé -65M)
- Pages détaillées par destination: description historique, expériences, prix et conseils
- Formulaire de réservation avec validation client et message de confirmation
- Chatbot flottant en bas à droite avec réponses simulées via `POST /api/chat`
- Transitions de pages douces via `app/template.tsx`

## Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir `http://localhost:3000`.
